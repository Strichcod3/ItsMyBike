# ItsMyBike IP-Symcon Module

IP-Symcon module to integrate **IT'S MY BIKE** GPS trackers
via the official IoT Venture API.

## Configuration

- Enter your phone number in the module configuration
- Phone number, SMS codes and tokens are stored locally
- No personal data is included in this repository
